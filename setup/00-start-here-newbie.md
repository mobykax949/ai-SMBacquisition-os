# Start Here — The Complete Newbie Walkthrough

*Never used Claude before? This is the only page you need to go from zero to your first deal analysis. Follow it top to bottom. Total time: about 20 minutes.*

You do not need to know how to code. You do not need a "developer" anything. If you can install an app and copy-paste, you can do this.

---

## Part 1 — Get Claude (5 minutes)

1. Go to **claude.com** and click **Sign up**. Choose the **Pro plan ($20/month)** — the free plan will not run these tools.
2. Go to **claude.com/download** and download the **Claude Desktop app** for your computer (Mac or Windows).
3. Open the app and sign in with the account you just made.

> **Mac users:** you need an Apple Silicon Mac (M1 chip or newer, made 2020 or later). If you have an older Intel Mac, the plugin will not run — email us and we will help you with the browser version.

**You are done with Part 1 when:** the Claude app is open on your computer and you are signed in.

---

## Part 2 — Turn on Cowork (2 minutes)

Cowork is the mode where Claude does real work and saves real files for you.

1. In the Claude app, find the mode switch and click **Cowork**.
2. Go to **Settings → Cowork** and set permissions to **"Ask before acting."** This makes Claude check with you before it does anything. Keep it here while you learn.

**You are done with Part 2 when:** you see "Cowork" selected at the top of the app.

---

## Part 3 — Install the Plugin (3 minutes, one time)

This is the step that gives you all 20 skills at once. You do NOT download them one by one.

1. In the Claude app, open **Settings**, then click **Plugins**.
2. Click **Add marketplace**.
3. In the box, paste this exactly:
   ```
   mobykax949/ai-SMBacquisition-os
   ```
   and confirm.
4. You will now see a plugin called **smb-wealth-builder**. Click **Install**.
5. **Important:** close your current chat and start a **brand-new Cowork conversation.** The skills only show up in a fresh chat.
6. In the new chat, type a single slash **/** — you should see a list of skills appear (journey-navigator, buy-box-builder, and more).

**You are done with Part 3 when:** typing **/** in a new chat shows the acquisition skills.

> **Nothing showed up?** 99% of the time it is because you did not start a NEW chat. Close the chat, open a fresh one, and type **/** again.

---

## Part 4 — Set Up Your Workspace (10 minutes)

Now make Claude yours. This one skill interviews you and sets everything up so Claude knows your situation in every future chat.

1. In a Cowork chat, paste this:
   ```
   Run the setup-my-workspace skill. Interview me one question at a time and set up my workspace.
   ```
2. Answer Claude's questions honestly — what you own, your budget, where you live, what you want. Use real numbers.
3. When it finishes, Claude will have created three files for you and will walk you through saving them into a **Project** so it remembers you forever.
4. Follow Claude's step-by-step Project instructions (it will tell you exactly what to click).

**You are done with Part 4 when:** Claude has told you your "journey stage" and which skill to run next.

---

## Part 5 — Run Your First Real Skill

Claude just told you your next step. For most people it is the **Journey Navigator** (confirms your strategy) then the **Buy-Box Builder** (defines what you are hunting). Just paste:

```
Run the journey-navigator skill on me.
```

From here, follow the lessons in order. Each one is a skill, and each skill produces a real file you can use.

---

## If you get stuck

- **The plugin will not install:** make sure you pasted `mobykax949/ai-SMBacquisition-os` exactly, with no spaces.
- **Skills do not appear:** start a NEW conversation. This is the #1 issue.
- **Claude forgets who I am each time:** you skipped the Project step in Part 4. Re-run setup-my-workspace and follow the Project setup.
- **Still stuck:** [community/support link] — send a screenshot and we will get you running.
