Getting Started with Claude for Acquisitions

Reading time: ~10 min. Outcome: Claude open, configured, and ready to run deal work.

Whether you are coming from ChatGPT or starting fresh, this gets Claude set up as your acquisition workspace. One subscription, a few settings, and you are operational.

1. Create your account and subscribe to Pro

  1. Go to claude.ai and create an account.
  2. Subscribe to Claude Pro ($20/month). Pro is required for the features this system depends on: Projects, larger context, and the ability to run longer agentic jobs without hitting limits mid-task.
  3. Install the Claude desktop app (Mac or Windows). The desktop app is where you will do the real work — it holds your Projects and skills in one place.

2. Three settings to turn on first

Setting Where Why it matters for deal work
Web search Settings → Features On by default. This is what powers on-market listing monitoring and market comps. Leave it on.
Projects Left sidebar Your persistent workspace. One Project per initiative (e.g. "OC Acquisitions") holds your buy box, method, and target notes so Claude always has context.
Analysis / file handling Settings → Features Lets Claude read spreadsheets, financials, and documents you upload during diligence.

3. The one rule that saves you

Files survive. Chat memory does not. A long conversation eventually compacts and Claude forgets the middle. So whenever you produce something worth keeping — a buy box, a target list, a valuation — save it to a file or into your Project. Treat the chat as a workbench, not a filing cabinet.

4. What "good" looks like when setup is done

  • Claude Pro active, desktop app installed.
  • A Project created for your acquisition work, with your buy box loaded.
  • Web search on.
  • You know where your files live.
